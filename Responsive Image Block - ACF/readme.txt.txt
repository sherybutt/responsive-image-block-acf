Follow the following steps to add gutenberg block to your wordpress site.

1 - Open the folder named "ACF - Fields" and import the file to your site through acf importer (ACF PRO VERSION REQUIRED).

2 - Open the folder "Templates Code >> function.php code" copy the code you find in the file and paste it at the end of your functions.php

3 - Open the folder "Templates Code >> block file code" copy the .php file and paste it in your wordpress directory with the hierarchy you defined in the code you past in function.php

4 - Thats it. Now search for the "Responsive image" block and use it.
